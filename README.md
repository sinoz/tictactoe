# TicTacToe
An online multiplayer TicTacToe game written in Scala. 

### Requirements
The project is compiled with Java 8 and uses SBT for build automation. The following dependencies are used across the specified modules:
- Client
  * LibGDX 1.9.4
    * Core
    * Platform (natives)
    * LWJGL-Backend
  * Netty 4.1.6
